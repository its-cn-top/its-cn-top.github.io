<p style="text-align: center;"><img alt="目前支持四种搜索引擎" src="https://images.gitee.com/uploads/images/2020/0512/181551_6587e200_7552126.png" /></p>

<p style="text-align: center;">在 V1.0.2 版本中，界面不设有搜索键，回车即可进行搜索。</p>
